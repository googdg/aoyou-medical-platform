## 技术栈选择

### 前端技术
- **基础**: HTML5 + CSS3 + JavaScript ES6+
- **框架**: 无框架（原生开发，轻量化）
- **UI 组件**: 自定义组件库（医学主题）
- **视频播放**: 自研播放器 + Video.js 备选
- **图表统计**: Chart.js 或 D3.js
- **微信集成**: 微信 JS-SDK

### 后端技术（接口对接）
- **API 协议**: RESTful API + JSON
- **认证方式**: JWT Token
- **数据格式**: JSON
- **文件上传**: 支持多种格式
- **实时通信**: WebSocket（可选）

### 数据存储
- **本地存储**: LocalStorage + IndexedDB
- **缓存策略**: Service Worker
- **离线支持**: PWA 技术
- **数据同步**: 增量同步机制

### 开发工具
- **版本控制**: Git
- **代码规范**: ESLint + Prettier
- **测试框架**: Jest + Cypress
- **构建工具**: Webpack 或 Vite
- **部署方式**: 静态文件部署
