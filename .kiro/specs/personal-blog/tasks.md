## 技术栈总结

### 前端技术
- **核心**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **架构**: 模块化设计，服务导向架构
- **功能**: 响应式设计，PWA支持，多语言，可访问性
- **优化**: 懒加载，缓存，性能监控

### 后端技术
- **服务器**: Node.js + Express.js
- **数据库**: SQLite3
- **认证**: JWT + bcryptjs
- **安全**: Helmet, CORS, 速率限制
- **文件处理**: Multer, 图片优化

### 部署和运维
- **容器化**: Docker + Docker Compose
- **进程管理**: PM2
- **反向代理**: Nginx (可选)
- **监控**: 日志记录，性能监控
