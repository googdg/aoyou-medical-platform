# 管理后台使用指南

## 🔐 登录信息

**默认登录账号：**
- 用户名：`admin`
- 密码：`admin123`

## 📝 功能说明

### 1. 主页内容管理
- 可以编辑网站标题、标语、描述
- 可以修改个人介绍、工作经历
- 可以更新社区活动和个人兴趣

### 2. 文章管理
- 创建新文章（支持中英文双语）
- 编辑现有文章
- 发布或保存为草稿
- 删除文章

### 3. 数据存储
- 目前使用浏览器本地存储（localStorage）
- 数据保存在浏览器中，清除浏览器数据会丢失
- 建议定期备份重要内容

## 🚀 使用步骤

### 访问管理后台
1. 打开 `admin/index.html` 文件
2. 使用默认账号登录

### 编辑主页内容
1. 点击左侧菜单的"主页内容"
2. 填写各个字段
3. 点击"保存更改"按钮

### 管理文章
1. 点击左侧菜单的"文章管理"
2. 点击"新建文章"创建文章
3. 填写标题、内容等信息
4. 选择"保存草稿"或"发布文章"

## ⚠️ 注意事项

1. **数据安全**：当前版本使用本地存储，请定期备份重要内容
2. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
3. **字段验证**：必填字段会有红色提示，请确保填写完整
4. **自动保存**：目前不支持自动保存，请及时手动保存

## 🔧 故障排除

### 保存失败
- 检查必填字段是否都已填写
- 刷新页面重试
- 检查浏览器控制台是否有错误信息

### 登录失败
- 确认用户名和密码正确
- 清除浏览器缓存重试

### 数据丢失
- 检查浏览器本地存储是否被清除
- 恢复之前的备份数据

## 📞 技术支持

如果遇到问题，请：
1. 检查浏览器控制台错误信息
2. 尝试刷新页面
3. 清除浏览器缓存重试

**提示**：这是一个演示版本的管理后台，适合个人使用和测试。生产环境建议使用完整的后端系统。
