# 🏥 奥友医学科普学习平台 - 演示版本

欢迎使用奥友医学科普学习平台演示版本！这是一个功能完整的医学教育平台，包含用户认证、视频学习、积分系统、移动端优化等核心功能。

## 🚀 快速开始

### 一键启动演示
```bash
./start-demo.sh
```

### 手动启动
```bash
# 1. 生成演示资源
./generate-placeholder-images.sh

# 2. 启动服务器
python3 -m http.server 3000
# 或
npm run dev

# 3. 访问演示
# 打开浏览器访问: http://localhost:3000/aoyou-medical-platform.html
```

## 🎯 演示功能

### 核心功能
- ✅ **用户认证系统** - 邀请码注册、登录验证
- ✅ **视频播放系统** - 多分类视频、播放控制、进度记录
- ✅ **积分等级系统** - 观看获得积分、8级等级制度、防刷机制
- ✅ **移动端优化** - 响应式设计、触摸交互、横竖屏适配
- ✅ **微信集成** - 分享功能、环境检测、自定义内容
- ✅ **数据持久化** - IndexedDB存储、离线同步、数据备份
- ✅ **性能优化** - 懒加载、预加载、Service Worker缓存
- ✅ **用户体验** - 加载状态、错误处理、引导教程
- ✅ **数据分析** - 行为追踪、性能监控、报表生成

### 医学分类
1. **临床医学** - 心血管疾病、糖尿病管理等
2. **基础医学** - 人体解剖学、生理学原理等
3. **外科手术** - 腹腔镜技术、骨科创伤等
4. **内科诊疗** - 呼吸系统、消化系统疾病等
5. **药学治疗** - 抗生素使用、心血管药物等
6. **前沿技术** - AI医学影像、精准医学等
7. **伦理法规** - 医学伦理、法律法规等
8. **继续教育** - 教育规划、循证医学等

9. `aoyou-medical-analytics.js` - 数据分析

## 🔧 自定义配置

### 添加新视频
```javascript
// 在 js/aoyou-medical-app.js 中添加
{
    id: 'custom_001',
    title: '您的视频标题',
    category: 'clinical',
    videoUrl: './videos/your-video.mp4',
    description: '视频描述',
    instructor: {
        name: '讲师姓名',
        title: '职称',
        hospital: '医院名称'
    }
}
```

## 🐛 故障排除

### 常见问题
1. **视频无法播放** - 检查视频文件格式和路径
2. **图片显示异常** - 确认图片文件存在且格式正确
3. **功能异常** - 查看浏览器控制台错误信息
4. **移动端问题** - 检查响应式CSS和触摸事件

### 调试方法
1. 打开浏览器开发者工具
