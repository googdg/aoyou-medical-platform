```
personal-blog/
├── admin/                  # 管理后台
│   ├── index.html         # 管理界面

│   ├── admin.css          # 管理界面样式

│   └── search-styles.css

│   ├── api-client.js      # API客户端
│   ├── blog-data.js       # 博客数据管理
│   ├── i18n.js           # 国际化
│   ├── search-engine.js   # 搜索引擎
│   ├── seo-optimizer.js   # SEO优化
│   └── sync-manager.js    # 内容同步
├── data/                  # 数据文件
│   ├── blog-posts.json    # 博客文章数据
│   └── i18n.json         # 多语言数据
├── test/                  # 测试文件
│   └── integration-test.js
├── .kiro/specs/          # 项目规范
│   └── personal-blog/
├── server.js             # 后端服务器
├── Dockerfile           # Docker配置
├── docker-compose.yml   # Docker Compose
├── deploy.sh           # 部署脚本
└── DEPLOYMENT.md       # 部署文档
```

## 🚀 部署方式

### 1. 本地开发
```bash
npm install
npm start
```

### 2. Docker部署
```bash
docker-compose up -d
```

### 3. 生产部署
```bash
./deploy.sh production
```
