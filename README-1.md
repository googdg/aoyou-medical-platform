# 🏥 奥友医学视频平台

一个专业的医学科普视频学习平台，采用邀请码验证机制，为特定用户群体提供高质量的医学教育内容。

## ✨ 核心特色

### 🔐 邀请码验证系统
- **会话级验证** - 每次新打开浏览器需要验证邀请码
- **数字邀请码** - 简单易用的数字验证码系统（任意数字即可）
- **一次验证，全程访问** - 验证成功后当前会话可观看所有视频
- **自动重置** - 关闭浏览器后验证状态自动清除

### 🎬 视频学习功能
- **YouTube风格界面** - 现代化的视频浏览体验
- **7个医学分类** - 临床医学、基础医学、药学治疗、影像诊断、检验医学、预防医学、伦理法规
- **视频详情页** - 完整的视频播放和信息展示
- **响应式设计** - 完美适配手机和平板设备

### 🎮 小游戏集合
- **小鬼消消乐** - 经典三消游戏，支持特殊道具和连击系统
- **黄金矿工** - 经典挖金游戏，多关卡挑战
- **坦克大战** - 经典坦克对战游戏，AI智能对手

### 🎯 用户体验
- **流畅动画效果** - 专业的交互体验
- **移动端优化** - 触摸交互和手势支持
- **加载状态管理** - 友好的加载提示和错误处理

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome 70+, Safari 12+, Firefox 65+）
- 支持ES6+的JavaScript环境
- 可选：Python 3.x（用于本地开发服务器）

### 本地开发

1. **克隆项目**
```bash
git clone https://github.com/你的用户名/你的仓库名.git
cd 项目目录
```

2. **启动开发服务器**
```bash
# 使用Python（推荐）
python3 -m http.server 3000

# 或使用Node.js
npx http-server -p 3000
```

3. **访问应用**
- **主平台**：`http://localhost:3000/aoyou-youtube-style.html`
- **视频详情页**：`http://localhost:3000/video-detail.html`
- **小鬼消消乐**：`http://localhost:3000/ghost-match3.html`
- **黄金矿工**：`http://localhost:3000/gold-miner-game.html`
- **坦克大战**：`http://localhost:3000/tank-battle-classic.html`

## 🛠️ 故障排除

### 常见问题及解决方案

#### 1. 邀请码弹窗不显示
- 检查是否已经验证过（当前浏览器会话）
- 在控制台运行 `resetVideoAccess()` 重置状态
- 使用隐身模式测试
- 访问 `invite-code-test.html` 进行功能测试

#### 2. 游戏无法正常运行
- 确保使用现代浏览器（Chrome 70+, Safari 12+）
- 检查浏览器控制台是否有错误信息
- 使用HTTP服务器访问（不是file://协议）
- 清除浏览器缓存后重试

#### 3. 移动端兼容性问题
- 检查viewport设置是否正确
- 测试触摸事件是否正常响应
- 确认屏幕方向适配
