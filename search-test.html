The search engine uses a weighted scoring system:

  • Title matches - Highest weight (10 points)
  • Tag matches - Medium-high weight (5 points)
  • Excerpt matches - Medium weight (3 points)
  • Content matches - Lower weight (1 point)
  • Language boost - Current language gets 20% bonus
  • Complete matches - All query terms matched gets 50% bonus
