# 🚀 WordPress 博客快速启动

## 最简单的方案：使用 Local by Flywheel

### 1. 下载安装 Local
- 访问：https://localwp.com/
- 点击 "Download for free"
- 选择你的操作系统（Mac/Windows/Linux）
- 下载并安装

### 2. 创建新站点
1. 打开 Local 应用
2. 点击 "Create a new site"
3. 输入站点名称：`my-blog`
4. 选择 "Preferred" 环境
5. 设置 WordPress 管理员信息：
   - 用户名：`admin`
   - 密码：`your-strong-password`
   - 邮箱：`your-email@example.com`

### 3. 启动并访问
1. 点击 "Start site" 启动站点
2. 点击 "WP Admin" 进入管理后台
3. 点击 "Open site" 查看博客首页

## 🎯 专注写作的配置

### 安装必要插件
在管理后台 → 插件 → 安装插件，搜索并安装：

1. **Classic Editor** - 使用熟悉的编辑器
2. **Yoast SEO** - SEO 优化
3. **Akismet Anti-Spam** - 垃圾评论过滤

### 选择简洁主题
在外观 → 主题中选择：
- **Twenty Twenty-Four** - 官方最新主题
- 或搜索 "blog" 找到专门的博客主题

### 基本设置
1. **设置 → 常规**
   - 站点标题：你的博客名称
   - 副标题：简短描述
   - 时区：Asia/Shanghai

2. **设置 → 阅读**
   - 首页显示：你的最新文章
   - 博客页面最多显示：10 篇文章

3. **设置 → 讨论**
   - 允许评论（如果需要互动）

## ✍️ 开始写作

### 发布第一篇文章
1. 点击 "文章 → 写文章"
2. 输入标题
3. 在编辑器中写内容
4. 选择分类或添加标签
5. 点击 "发布"

### 文章管理
- **草稿**：未发布的文章
- **已发布**：公开可见的文章
- **私密**：只有你能看到
- **定时发布**：设置未来发布时间

### 分类和标签
- **分类**：文章的主要类别（如：技术、生活、思考）
- **标签**：文章的关键词（如：JavaScript、旅行、读书）
