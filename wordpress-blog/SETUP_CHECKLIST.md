# ✅ WordPress 博客设置检查清单

恭喜你已经安装了 WordPress！现在让我们完成初始设置，让你的博客更适合写作。

## 🎯 基础设置

### 1. 网站基本信息
- [ ] **设置 → 常规**
  - [ ] 站点标题：输入你的博客名称
  - [ ] 副标题：简短描述你的博客
  - [ ] WordPress 地址 (URL)：确认正确
  - [ ] 站点地址 (URL)：确认正确
  - [ ] 管理员邮箱地址：输入你的邮箱
  - [ ] 时区：选择 `UTC+8` 或 `Asia/Shanghai`
  - [ ] 日期格式：选择你喜欢的格式
  - [ ] 时间格式：选择 24 小时制或 12 小时制
  - [ ] 一周开始于：选择星期一或星期日

### 2. 阅读设置
- [ ] **设置 → 阅读**
  - [ ] 您的主页显示：选择 `您的最新文章`
  - [ ] 博客页面至多显示：`10` 篇文章
  - [ ] 聚合器中最多显示：`10` 篇文章
  - [ ] 对于聚合器中的每篇文章，显示：选择 `摘要`
  - [ ] 搜索引擎可见性：确保**未勾选**（除非你不想被搜索引擎收录）

### 3. 讨论设置
- [ ] **设置 → 讨论**
  - [ ] 默认文章设置：
    - [ ] 尝试通知文章中链接的所有博客（可选）
    - [ ] 允许其他博客发送链接通知（可选）
    - [ ] 允许他人在新文章上发表评论（根据需要）
  - [ ] 其他评论设置：
    - [ ] 评论者必须填入姓名和邮箱地址
    - [ ] 用户必须注册并登录才可以发表评论（可选）
    - [ ] 自动关闭发布超过 `14` 天的文章的评论功能
  - [ ] 评论审核：
    - [ ] 评论必须经人工批准
    - [ ] 评论者先前须有评论通过了审核

### 4. 媒体设置
- [ ] **设置 → 媒体**
  - [ ] 缩略图尺寸：`150 x 150`
  - [ ] 中等尺寸：`300 x 300`
  - [ ] 大尺寸：`1024 x 1024`
  - [ ] 将我上传的文件按年月整理到文件夹中：**勾选**

### 5. 固定链接设置
- [ ] **设置 → 固定链接**
  - [ ] 选择 `文章名` 或 `自定义结构`：`/%postname%/`
  - [ ] 这样 URL 更友好，有利于 SEO

## 🔌 必装插件

### 写作增强插件
- [ ] **Classic Editor**
  - 安装：插件 → 安装插件 → 搜索 "Classic Editor"
  - 用途：使用熟悉的经典编辑器界面

- [ ] **TinyMCE Advanced**
  - 用途：增强编辑器功能，添加更多格式选项

### SEO 优化插件
- [ ] **Yoast SEO**
  - 安装：插件 → 安装插件 → 搜索 "Yoast SEO"
  - 用途：SEO 优化，提高搜索引擎排名

### 安全和维护插件
- [ ] **Akismet Anti-Spam**
  - 通常已预装，需要激活并获取 API 密钥
  - 用途：过滤垃圾评论

- [ ] **UpdraftPlus**
  - 用途：自动备份网站和数据库

### 性能优化插件
- [ ] **WP Super Cache**
  - 用途：缓存页面，提高网站速度
