# ✍️ WordPress 写作完全指南

这份指南将帮你充分利用 WordPress 的写作功能，创作出优质的博客内容。

## 🚀 快速开始写作

### 创建新文章
1. 登录 WordPress 后台
2. 点击 **文章 → 写文章**
3. 输入文章标题
4. 在编辑器中写内容
5. 点击 **发布**

### 编辑器选择
- **古腾堡编辑器**（默认）：块编辑器，适合富媒体内容
- **经典编辑器**：传统编辑器，类似 Word 界面

## 📝 文章结构最佳实践

### 标题优化
```
好标题示例：
✅ "5个提高工作效率的时间管理技巧"
✅ "初学者必知的 Python 编程基础"
✅ "我的日本旅行攻略：东京3日游完整指南"

避免的标题：
❌ "一些想法"
❌ "今天的文章"
❌ "关于编程"
```

### 文章结构模板
```markdown
# 吸引人的标题

## 引言
- 简短介绍文章主题
- 说明读者能获得什么

## 主要内容
### 要点1：具体标题
- 详细说明
- 举例或数据支持

### 要点2：具体标题
- 详细说明
- 举例或数据支持

### 要点3：具体标题
- 详细说明
- 举例或数据支持

## 总结
- 重点回顾
- 行动建议
- 相关文章推荐
```

## 🎨 内容格式化

### 使用标题层级
- **H1**：文章标题（自动生成）
- **H2**：主要章节
- **H3**：子章节
- **H4-H6**：更细分的内容

### 文本格式
- **粗体**：重要内容
- *斜体*：强调或引用
- `代码`：技术术语或代码片段
- > 引用：引用他人观点

### 列表使用
**有序列表**（步骤、排名）：
1. 第一步
2. 第二步
3. 第三步

**无序列表**（要点、特性）：
- 要点一
- 要点二
- 要点三

## 📷 图片和媒体

### 图片最佳实践
1. **尺寸优化**
   - 宽度：800-1200px
   - 文件大小：< 500KB
   - 格式：JPG（照片）、PNG（图标）

2. **SEO 优化**
   - 文件名：使用描述性名称
   - Alt 文本：描述图片内容
   - 标题：简短说明

3. **插入图片**
   - 点击编辑器中的 "添加媒体"
   - 上传或选择现有图片
   - 设置对齐方式和大小

### 其他媒体
- **视频**：支持 YouTube、Vimeo 嵌入
- **音频**：上传 MP3 文件
- **文档**：PDF、Word 文档下载

## 🏷️ 分类和标签

### 分类（Categories）
- **用途**：文章的主要类别
- **特点**：层级结构，一篇文章通常属于一个分类
- **示例**：
  - 技术
    - 前端开发
    - 后端开发
  - 生活
    - 旅行
    - 美食

### 标签（Tags）
- **用途**：文章的关键词
- **特点**：扁平结构，一篇文章可以有多个标签
- **示例**：JavaScript, React, 教程, 初学者

### 使用建议
```
文章：《React 初学者入门教程》
分类：技术 → 前端开发
标签：React, JavaScript, 教程, 初学者, 前端
```

## 🔍 SEO 优化写作

### 关键词策略
1. **主关键词**：文章的核心主题
2. **长尾关键词**：更具体的搜索词
3. **相关关键词**：主题相关的词汇

### SEO 检查清单
- [ ] 标题包含主关键词
- [ ] 文章开头100字内出现关键词
- [ ] 使用 H2、H3 标题分段
- [ ] 图片设置 Alt 文本
- [ ] 文章长度 > 300 字
- [ ] 内部链接到相关文章
- [ ] 设置文章摘要

### 使用 Yoast SEO 插件
1. 安装 Yoast SEO 插件
2. 在文章编辑页面下方找到 SEO 设置
3. 设置焦点关键词
4. 优化 SEO 标题和描述
5. 查看可读性分析

## ⏰ 发布策略

### 文章状态
- **草稿**：未完成的文章
- **待审核**：等待审核的文章
- **已发布**：公开可见的文章
- **私密**：只有你能看到
- **受密码保护**：需要密码才能查看

### 定时发布
1. 在发布框中点击 "立即发布" 旁的 "编辑"
2. 设置发布日期和时间
3. 点击 "确定"
4. 点击 "安排发布"

### 发布频率建议
- **新博客**：每周 1-2 篇
- **成熟博客**：每周 2-3 篇
- **专业博客**：每天 1 篇

## 🔄 文章管理

### 编辑已发布文章
1. 文章 → 所有文章
2. 找到要编辑的文章
3. 点击 "编辑"
4. 修改后点击 "更新"

### 文章版本管理
- WordPress 自动保存文章修订版本
- 可以恢复到之前的版本
- 在编辑页面右侧查看 "修订版本"

### 批量操作
- 选择多篇文章
- 选择批量操作：删除、编辑、移至回收站
- 点击 "应用"

## 💬 评论管理

### 评论设置
1. 设置 → 讨论
2. 配置评论规则
3. 设置审核要求

### 回复评论
1. 评论 → 所有评论
2. 找到要回复的评论
3. 点击 "回复"
4. 输入回复内容

### 评论审核
- **批准**：公开显示评论
- **垃圾评论**：标记为垃圾
- **回收站**：删除评论

## 🎯 写作技巧

### 提高可读性
1. **短段落**：每段 2-3 句话
2. **简单句式**：避免复杂的长句
3. **过渡词**：使用连接词让文章流畅
4. **视觉元素**：使用列表、引用、图片

### 吸引读者
1. **开头抓人**：用问题、故事或数据开头
2. **价值导向**：明确告诉读者能获得什么
3. **互动元素**：提问、邀请评论
4. **行动号召**：引导读者下一步行动

### 保持一致性
- **写作风格**：保持个人特色
- **发布频率**：规律更新
- **主题聚焦**：围绕核心主题
- **质量标准**：维持内容质量
