# 自定义插件目录

将你的自定义 WordPress 插件放在这个目录中。

## 使用方法

1. 将插件文件夹复制到此目录
2. 在 WordPress 后台 -> 插件 中激活插件

## 推荐插件

### 写作增强
- **Classic Editor**: 使用经典编辑器
- **TinyMCE Advanced**: 增强编辑器功能
- **WP Editor.md**: Markdown 编辑器

### SEO 优化
- **Yoast SEO**: 全面的 SEO 解决方案
- **Rank Math**: 现代 SEO 插件
- **All in One SEO**: 简单易用的 SEO 插件

### 性能优化
- **WP Super Cache**: 静态缓存
- **W3 Total Cache**: 全面缓存解决方案
- **Autoptimize**: CSS/JS 优化

### 安全防护
- **Wordfence Security**: 安全防护
- **Sucuri Security**: 恶意软件扫描
- **iThemes Security**: 全面安全解决方案

### 备份
- **UpdraftPlus**: 自动备份
- **BackWPup**: 完整备份解决方案

### 其他实用插件
- **Akismet**: 垃圾评论过滤
- **Contact Form 7**: 联系表单
- **WP Mail SMTP**: 邮件发送
- **Duplicate Post**: 复制文章/页面
